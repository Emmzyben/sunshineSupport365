# SunshineSupport365

**SunshineSupport365** is a premier staffing and recruitment solutions provider specializing in healthcare, childcare, education, cleaning, and advocacy services across the UK.

## Features

### 🌍 Multi-Language Support
- **Google Translate Integration**: Seamlessly switch between multiple languages to cater to a diverse clientele.
- Supported Languages: English, Spanish, French, Chinese, and many more.

### 📋 Comprehensive Service Categories
We offer staffing solutions across a wide range of sectors:
- **Healthcare**: Nurses, Support Workers, Care Workers, Care Managers, Domiciliary Care
- **Education**: Nursery Assistants, Teaching Assistants
- **Professional Services**: Translators, Interpreters, Cleaners, Advocacy Services, Charity Workers
- **Childcare**: Nannies

### 🧩 Sector-Specific Pages
Each service category has dedicated landing pages with detailed information:
- Care Workers
- Care Managers
- Social Workers
- Support Workers
-Translators
-Nurses
-Charity workers
-Nannies
-Professional Cleaners
-Teaching Assistants
-Advocacy Services

### 📱 Responsive Design
- **Mobile-First**: Fully responsive layout optimized for all devices (desktops, tablets, smartphones)
- **Side Navigation**: Collapsible side menu for easy navigation on mobile devices

### 💼 User Engagement
- **Bookings**: Dedicated booking system for clients to request staffing services
- **Hiring Portal**: "We're Hiring" section with direct email integration for job applicants
- **Contact Forms**: Easy-to-use contact forms for inquiries and support

## 🚀 Performance Features

- **Lazy Loading**: Optimized image loading to improve page load times
- **Clean Code**: Well-structured HTML and CSS with semantic markup
- **Accessibility**: WCAG 2.1 compliant design with proper ARIA labels

## 🛠️ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/SunshineSupport365.git
   ```
2. Open `index.html` in your web browser:
   ```bash
   open index.html
   ```
